# Submission Portal (Simple)
Đây là bộ file để chạy một trang web nhận bài/ảnh từ học sinh.

## Cấu trúc
- public/ : frontend (index.html, admin.html, styles.css, app.js)
- uploads/ : nơi lưu file đã upload
- submissions.json : dữ liệu metadata (tên, thời gian, file, ghi chú)
- server.js : server Node.js + Express
- package.json : dependency
